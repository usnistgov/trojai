Minimal Example¶

NIST has developed a minimal working example showing a submission to the TrojAI NIST Test and Evaluation server. This ‘solution’ loads the model file, inferences 10 random tensors, and then writes a random number to the output file.

https://github.com/usnistgov/trojai-example

Included in the repository README.md is instructions for:

  1. confirming the code works on your hardware, and

  2. packaging the solution into a Singularity container.
